# Agents

Guidelines for AI agents working in this repository.

## READMEs

Keep README.md files up to date when making changes. If you add, remove, or rename files in a directory, update the corresponding README to reflect the current state. Every `operator/` and `instance/` folder, every component, and every top-level directory should have an accurate README.

## Architecture Decision Records

When a decision is made about how something should be structured, deployed, or configured, create an ADR in `docs/decisions/` following the MADR template in `docs/decisions/adr-template.md`. Use the naming convention `NNNN-title-with-dashes.md` with the next sequential number.

## Commits

Do not add `Co-Authored-By` lines or any AI attribution to commit messages.
